# Application Routes Summary

## Public Routes (Pages)

### Main Routes
- **`/`** - Landing page / Login / Portal (role-based redirect)
  - Shows landing page for unauthenticated users
  - Shows login form
  - Redirects to role-specific portal after login

### Role-Based Portal Routes
- **`/employee`** - Employee Portal
  - Dashboard
  - Apply for Leave
  - Leave Balances
  - Leave History
  - Notifications
  - Profile
  - Documents

- **`/hr`** - HR Portal
  - Dashboard
  - Staff Management
  - Leave Management
  - Leave Policies
  - Holidays
  - Leave Templates
  - Reports & Analytics

- **`/manager`** - Manager Portal
  - Dashboard
  - Leave Approvals
  - Team View

- **`/admin`** - Admin Portal
  - Dashboard
  - User Management
  - Audit Logs
  - System Settings

### Authentication Routes
- **`/reset-password`** - Password Reset Page

### System Routes (Next.js)
- **`/_not-found`** - 404 Not Found page (auto-generated by Next.js)

## API Routes

### Authentication API
- `POST /api/auth/login` - User login
- `POST /api/auth/logout` - User logout
- `GET /api/auth/me` - Get current user
- `POST /api/auth/register` - Register new user

### Staff API
- `GET /api/staff` - Get all staff members
- `GET /api/staff/[id]` - Get staff member by ID
- `POST /api/staff` - Create staff member
- `PATCH /api/staff/[id]` - Update staff member
- `POST /api/staff/[id]/assign-manager` - Assign manager to staff
- `POST /api/staff/bulk-assign-manager` - Bulk assign manager

### Leave Requests API
- `GET /api/leaves` - Get all leave requests
- `GET /api/leaves/[id]` - Get leave request by ID
- `POST /api/leaves` - Create leave request
- `PATCH /api/leaves/[id]` - Update leave request (approve/reject)
- `POST /api/leaves/[id]/cancel` - Cancel leave request
- `POST /api/leaves/bulk` - Bulk operations
- `POST /api/leaves/calculate-days` - Calculate leave days

### Leave Balances API
- `GET /api/balances` - Get all leave balances
- `GET /api/balances/[staffId]` - Get balance for specific staff

### Leave Policies API
- `GET /api/leave-policies` - Get all leave policies
- `GET /api/leave-policies/[id]` - Get policy by ID
- `POST /api/leave-policies` - Create policy
- `PATCH /api/leave-policies/[id]` - Update policy

### Leave Templates API
- `GET /api/leave-templates` - Get all templates
- `GET /api/leave-templates/[id]` - Get template by ID
- `POST /api/leave-templates` - Create template
- `PATCH /api/leave-templates/[id]` - Update template

### Holidays API
- `GET /api/holidays` - Get all holidays
- `GET /api/holidays/[id]` - Get holiday by ID
- `POST /api/holidays` - Create holiday
- `PATCH /api/holidays/[id]` - Update holiday
- `DELETE /api/holidays/[id]` - Delete holiday

### Notifications API
- `GET /api/notifications` - Get all notifications
- `GET /api/notifications/[id]` - Get notification by ID
- `POST /api/notifications/mark-read` - Mark notifications as read
- `POST /api/notifications/send-announcement` - Send announcement

### Audit Logs API
- `GET /api/audit-logs` - Get audit logs
- `GET /api/audit-logs/[id]` - Get audit log by ID

### Approvals API
- `POST /api/approvals/reminders` - Send approval reminders

### Monitoring API
- `GET /api/monitoring/health` - System health check

### Realtime API
- `GET /api/realtime` - Realtime connection endpoint
