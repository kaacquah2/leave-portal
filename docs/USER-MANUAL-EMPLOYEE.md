## üìù Applying for Leave

### Step-by-Step Guide

1. **Navigate to "Apply for Leave"**
   - Click "Apply for Leave" from the dashboard or sidebar

2. **Select Leave Type**
   - Choose from: Annual, Sick, Unpaid, Special Service, Training, Study, Maternity, Paternity, or Compassionate
   - **Note**: Available balance is shown for each leave type

3. **Select Dates**
   - Choose start date
   - Choose end date
   - System automatically calculates days (excluding holidays)

4. **Enter Reason**
   - Provide a clear reason for your leave request
   - Be specific and professional

5. **Add Attachments (Optional)**
   - Upload supporting documents:
     - Medical reports (for Sick/Maternity leave)
     - Training letters (for Training leave)
     - Official memos (for Special Service leave)

6. **Review and Submit**
   - Review all information
   - Check available balance
   - Click "Submit Leave Request"

### Important Notes

- **Balance Check**: The system shows your available balance before submission
- **Holiday Exclusion**: Holidays are automatically excluded from leave days
- **Overlapping Leaves**: You cannot submit overlapping leave requests
- **Insufficient Balance**: If balance is insufficient, contact HR for assistance

- **Available**: Days you can use

## üìÖ Leave History

### Viewing Your Leave History

1. Navigate to "Leave History" from the sidebar
2. See all your leave requests with:
   - Leave type and dates
   - Number of days
   - Status (Pending, Approved, Rejected, Cancelled)
   - Approval information
   - Comments from approvers

### Leave Statuses

- **Pending**: Awaiting approval
- **Approved**: Leave has been approved
- **Rejected**: Leave was rejected (see comments for reason)
- **Cancelled**: You cancelled the leave request

### Actions Available

- **View Details**: Click on any leave to see full details
- **Download Approval Letter**: For approved leaves
- **Cancel Leave**: Cancel pending or approved leaves (balance will be restored)

## ‚ùì Troubleshooting

### Common Issues

#### Cannot Submit Leave Request

**Problem**: "Insufficient leave balance" error

**Solution**:
- Check your available balance
- Contact HR if you need to request leave despite insufficient balance
- Consider using Unpaid leave if applicable

#### Cannot See Leave Request

**Problem**: Leave request not appearing

**Solution**:
- Refresh the page
- Check if you're looking at the correct date range
- Contact HR if issue persists

#### Balance Not Updated

**Problem**: Balance doesn't reflect approved leave

**Solution**:
- Wait a few moments (system updates automatically)
- Refresh the page
- Contact HR if balance is incorrect

#### Cannot Cancel Leave

**Problem**: Cancel button not working

**Solution**:
- Only pending or approved leaves can be cancelled
- Rejected leaves cannot be cancelled
- Contact HR if you need to cancel a rejected leave
