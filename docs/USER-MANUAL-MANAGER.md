## ‚úÖ Approving Leave Requests

### Accessing Pending Approvals

1. Navigate to "Approve Leaves" from the sidebar
2. View all pending leave requests from your team

### Approval Process

1. **Review Leave Request**
   - Check leave type and dates
   - Verify available balance
   - Read reason and attachments

2. **Make Decision**
   - **Approve**: Click "Approve" button
   - **Reject**: Click "Reject" button
   - **Add Comments**: Provide feedback (required for rejection)

3. **Submit Decision**
   - System automatically deducts balance on approval
   - Employee receives notification
   - Leave appears in team calendar

### Important Notes

- **Balance Validation**: System prevents approval if balance is insufficient
- **Multi-Level Approval**: Some leaves require HR approval after manager approval
- **Comments**: Always add comments, especially for rejections
- **Delegation**: You can delegate approvals if unavailable

## üë§ Delegation

### Delegating Approvals

1. Navigate to "Delegation" from the sidebar
2. Set up delegation:
   - Select delegate (another manager or HR)
   - Set start and end dates
   - Select leave types (or all types)
   - Add notes

3. **Delegation Rules**:
   - Delegates can approve on your behalf
   - You can revoke delegation anytime
   - Delegation expires automatically

## ‚ùì Troubleshooting

### Common Issues

#### Cannot See Team Members

**Problem**: No team members showing

**Solution**:
- Contact HR to assign staff to you as manager
- Verify your manager assignment in system

#### Cannot Approve Leave

**Problem**: Approval button not working or error message

**Solution**:
- Check if you're the assigned approver
- Verify leave is still pending
- Check if balance is sufficient (system prevents approval if insufficient)
- Contact HR if issue persists

#### Balance Error on Approval

**Problem**: "Insufficient balance" error when approving

**Solution**:
- This is expected behavior - system prevents approval if employee has insufficient balance
- Contact HR if this leave should be approved despite insufficient balance
- HR can override if necessary
